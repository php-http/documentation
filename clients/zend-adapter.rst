Zend Adapter
============

An HTTPlug adapter for the Zend HTTP client.

Installation
------------

To install the Zend adapter, which will also install Zend itself (if it was
not yet included in your project), run:

.. code-block:: bash

    $ composer require php-http/zend-adapter
